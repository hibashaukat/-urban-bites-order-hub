# 🍔 Urban Bites - Smart Digital Menu & WhatsApp Ordering Hub

An ultra-modern, mobile-first Digital Menu and Instant WhatsApp Ordering Web Application designed for modern fast-casual restaurants. Features a sleek interactive UI, instant category filtering, dynamic cart management, and seamless direct-to-WhatsApp order routing.

---

## ✨ Key Features

*   **📱 Mobile-First App Vibe:** Optimized heavily for mobile viewports with premium UI/UX, smooth transitions, and glassmorphic elements.
*   **⚡ Sticky Smart Category Selector:** A horizontal scrolling interactive category bar (Burgers, Pizzas, Starters, etc.) with active-state transitions for effortless navigation.
*   **🛒 Dynamic Floating Cart Island:** A real-time updating bottom checkout dock that calculates quantities and total prices on the fly with micro-animations.
*   **💬 Direct WhatsApp Order Routing:** Automagically constructs a pre-formatted, clean message detailing the items ordered, quantities, and total amount, sending it straight to the restaurant owner's WhatsApp number.
*   **🔥 Promotional Badges:** Visually appealing tags like `BEST SELLER`, `HOT`, and `CHEF'S CHOICE` to drive customer engagement.
*   **🌐 Optimized Asset Loading:** Integrated with a global CDN architecture to ensure high-performance image rendering across all networks.

---

## 🛠️ Tech Stack

*   **Frontend Library:** React.js (built with Vite for lightning-fast bundling)
*   **Styling:** Tailwind CSS (for modern utility-first, responsive layouts)
*   **Icons:** Lucide React
*   **Deployment:** Vercel

## 🚀 Getting Started Locally

Follow these steps to set up and run the project on your local machine:

### 1. Clone the repository
```bash
git clone [https://github.com/hibashaukay/urban-bites-demo.git](https://github.com/hibashaukat/urban-bites-demo.git)
cd urban-bites-demo

2. Install Dependencies
Bash
npm install
3. Run Development Server
Bash
npm run dev
Open http://localhost:5173 in your browser to see the live app.

📦 Deployment
This project is automatically configured for production deployment on Vercel with continuous integration (CI/CD) through GitHub. Any push to the main branch will trigger an automatic production build.
